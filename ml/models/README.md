# Model Artifacts

Place trained model artifacts here only if they are small enough for the repo.

Expected CDX-006 outputs after the Colab run:

- `best.pt`
- `best.onnx`
- confusion matrix image or a pointer to the run directory
- `ml/reports/eval_metrics.json`

If weights are too large for git, store them in a GitHub release or Drive folder and update `ml/MODELCARD.md` with the exact link.
